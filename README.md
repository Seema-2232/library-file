# Smart Library Management System (AI Based)

A GUI-based Smart Library Management System built using Python and Tkinter.

## Features

- Add Books
- View Books
- File-based Persistent Storage
- Keyword-Based Recommendation
- Related Book Suggestion
- NLP Matching using TF-IDF
- Cosine Similarity Ranking

## Technologies Used

- Python
- Tkinter
- Scikit-learn
- TF-IDF Vectorization
- Cosine Similarity

## How to Run

1. Install dependencies:
   pip install scikit-learn

2. Run:
   python main.py
